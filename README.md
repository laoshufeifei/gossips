# Gossips

gossips 是一个记录、学习、实现常用数据结构和算法的工程。

项目定位是个学习型的 demo，不一定用于生产，所以很多实现没有支持泛型。

造轮子（不是重新发明轮子）是为了更好的熟悉数据结构和算法。

参考项目: 

[gods](https://github.com/emirpasic/gods)
